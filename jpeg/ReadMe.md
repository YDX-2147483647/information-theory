```mermaid
flowchart LR
    Compressed --> entropy["Entropy Decoder"]

    subgraph Decoder

    end

    table -.-> entropy
```

Entropy Encoder:

- AC: Run length Encoding.
- DC: Differential Pulse Code Modulation.
